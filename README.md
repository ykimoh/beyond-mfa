# Beyond, Original Research and Prototype (SVA, 2023)

Standalone hosting for *Beyond: A tool that helps put inclusive and accessible design into action*, an original research project and integrated prototype system for inclusive cross-platform UX design by Young Kim. Completed in 2023 as the culminating project of the Master of Fine Arts in Interaction Design program at the School of Visual Arts.

Live at [ykimoh.github.io/beyond-mfa](https://ykimoh.github.io/beyond-mfa/). Indexed from [ykimoh.github.io](https://ykimoh.github.io).

## What's here

- `index.html`, landing page with abstract, document links, and citation
- `styles.css`, site styles (matches the hub)
- `Beyond_Process_Book.pdf`, full 66-page process book
- `Beyond_Presentation.pdf`, companion summary deck

Plain HTML and CSS, no build step. Served via GitHub Pages from the repo root.

## Accessibility

WCAG 2.1 AA target. Semantic landmarks, skip-link, visible focus indicators, system font stack, dark-mode via `prefers-color-scheme`.
